Installation

Composer

Install using composer

$ composer require damienpirsy/teamwork ~1.0

Laravel Integration

Add the Service Provider to the providers array:


'providers' => [
    ...
    Damienpirsy\Teamwork\TeamworkServiceProvider::class,
],

Add the Teamwork Fa√ßade to the aliases array:


'aliases' => [
    ...
    'Teamwork' => Damienpirsy\Teamwork\Facades\Teamwork::class,
],

Lastly, create a 'teamwork' array in the config/services.php file:


'teamwork' => [
    'key'  => 'YourSecretKey',
    'url'  => 'YourTeamworkUrl' #your teamwork url, ex. http://[companyname].teamwork.com'
],

Please refer to the Teamwork website for instruction on how to request your API credentials.

Non Laravel install

The package can be used outside Laravel by loading the required classes.


require "vendor/autoload.php";

use GuzzleHttp\Client as Guzzle;
use Damienpirsy\Teamwork\Client;
use Damienpirsy\Teamwork\Factory as Teamwork;

$client     = new Client(new Guzzle, 'YourSecretKey', 'YourTeamworkUrl');
$teamwork   = new Teamwork($client);
