Usage & Documentation

Usage

Façade accessor

If you are using the Façade with Laravel you can easily access Teamwork like this


$people = Teamwork::people()->all();

Restful Trait

Most endpoints implements a commont Restful Trait for basic CRUD operations and records fetching. Just change method() with the corresponding one:


// returns all the records, with NO options passed
$data = Teamwork::method()->all();

// Convenient method to paginates the records, which is the same as using all(['page' => 10, 'pageSize' => 200])
// The $pageSize param is OPTIONAL and defaults to 200
$data = Teamwork::method()->paginate(10, $pageSize)

// returns the record based on the passed ID
$data = Teamwork::method($id)->find()

// creates a new element
// @param array $data
// POST
Teamwork::method()->create($data);

// updates an element
// @param array $data
// PUT
Teamwork::method($id)->update($data);

// Deletes a record
// DELETE
Teamwork::method($id)->delete();

Response

All returned responses are arrays containing the original Teamwork response along with the provided x-records, x-pages and x-page pagination headers sent, to allow for an easier pagination of records for the consumer. Example of response:


array:5 [
  "people" => array:8 [...]
  "STATUS" => "OK"
  "records" => "8"
  "pages" => "1"
  "page" => "1"
]

Documentation

Account

Returns information about the account. See official API section for further info.


// Get account details
// GET /account.json
$data = Teamwork::account()->details();

// Returns details about the Teamwork account and some basic details of the current user being authenticated.
// GET /authenticate.json
$data = Teamwork::account()->authenticate();

Activity

Get latest activity across all projects, ordered chronologically:

Accepted params:

  • maxItems (numeric, default=60, Max=200)
  • onlyStarred (boolean, default=no)

// GET /latestActivity.json
// Lists the latest activity across all projects ordered chronologically
$data = Teamwork::activity()->latest(['maxItems' => 10]);
// Get only starred activity by the account your are using.
$data = Teamwork::activity()->latest(['onlyStarred' => 1]);

Delete an activity entry:

 
// Delete a specific activity by ID
$data = Teamwork::activity($id)->delete();

Billing

Invoices

Implements the RestuflTrait, the all() method is overridden.


// get all invoices. 
// GET /invoices.json
// Optional params type, projectStatus, page
$data = Teamwork::invoices()->all();

//  Mark a specific invoice as complete
//  PUT /invoices/:id/complete.json
Teamwork::invoices($id)->complete();

// Mark a specific invoice as not complete
//  PUT /invoices/:id/uncomplete.json
Teamwork::invoices($id)->uncomplete();

// Get a list of valid currency codes
// GET /currencycodes.json
Teamwork::invoices()->currencies();

Milestones

Implements the RestuflTrait with the milestones() method. The all() and find() methods are overridden.


// Get all milestones in Teamwork.
$data = Teamwork::milestones()->all();

// Get all milestones and get progress of each milestone.
$data = Teamwork::milestones()->all(['getProgress' => 'true']);

// Find a specific milestone by id
$data = Teamwork::milestones($id)->find();

// Find milestone by ID with tasks, task lists and progress.
$data = Teamwork::milestones($id)->find([
   'getProgress' => 'true',
   'showTaskLists' => 'true',
   'showTasks' => 'true'
]);

// Marks the specified milestone as complete
// PUT /milestones/{id}/complete.json
Teamwork::milestones($id)->complete();

// Marks the specified milestone as uncomplete
// PUT /milestones/{id}/uncomplete.json
Teamwork::milestones($id)->uncomplete();

People

Implements the RestuflTrait with the people() method. The all() method is overridden.


// Get a person by email address
$data = Teamwork::people()->all(['emailaddress' => 'test@awesome.com']);

// Returns the details for the account you're currently logged in with
$data = Teamwork::people()->me();

// Retrieve a API Keys for all people on account
$data = Temwork::people()->apiKeys();

Status


// Get all peoples' status
$data = Teamwork::status()->all();

// Get my status
$data = Teamwork::status()->mine();

// Get a person's status
$data = Teamwork::status()->persons($id);

// create a person's status
Teamwork::status()->createPersons($person_id, $data);

// create my status
Teamwork::status()->createMine($data);

// Update a status by ID
Teamwork::status($id)->update($data);

// Update my status by ID
Teamwork::status($id)->updateMine($data);

// Update a specific person's status by ID
Teamwork::status($id)->updatePersons($person_id, $data);

// Deletes a status by ID
Teamwork::status($id)->delete();

// Deletes my status by ID
Teamwork::status($id)->deleteMine();

// Deletes a person's status by ID
Teamwork::status($id)->deletePersons($person_id);

Permissions


// Add a new user to a project
Teamwork::project($id)->addUser($user_id);

// Add/Remove multiple people to/from a project
Teamwork::project($id)->addRemoveUsers($data);

// Remove a user from a project
Teamwork::project($id)->removeUser($user_id);

// Get a user permissions on a project
$data = Teamwork::project($id)->permission($user_id);

// Set a user permissions on a project
Teamwork::project($id)->setPermission($user_id, $data);

Tasks


// Get all tasks.
Teamwork::task()->all();

// You can also filter tasks by many different parameters that are
// listed here in the Teamwork developers docs. This example
// shows how to filter by tasks that are overdue and then
// order them by date.
Teamwork::task()->all(['filter' => 'overdue', 'sort' => 'duedate']);

// Retrieve a task by id
Teamwork::task($id)->find();

// Retrieve a task by id and exclude files and subtasks.
Teamwork::task($id)->find(['getFiles' => 'false', 'nestSubTasks' => 'true]);

// Complete and uncomplete a task.
Teamwork::task($id)->complete();
Teamwork::task($id)->uncomplete();


// Get all reminders on a task
$data = Teamwork::task($id)->reminders();
// Create a reminder on a task
Teamwork::task($id)->setReminders($data);
// Updates a reminder on a task
Teamwork::task($id)->updateReminder($id_reminder, $data);
// Deletes a reminder on a task
Teamwork::task($id)->deleteReminder($id_reminder);

Tags

Fully implements the RestuflTrait with the tag() method.

Additionaly, the TagsTrait is implemented by several resources to access the corresponding tags. Example:


// Project tags
$data = Teamwork::project($id)->tags();
// Update all project tags
Teamwork::project($id)->updateTags($data);
